# 📍 Loca Phone Location Tracker Web

**A complete, sleek, fully functional web application** replicating the popular Android app “Phone Location Tracker — Find Family, Friends”.

## ✨ Features
- 📍 **Interactive Live Map** – Leaflet-powered OpenStreetMap + Satellite toggle  
- 👥 **Real-time Friends Tracking** – Fake live-moving family & friends markers  
- 🛡️ **Geofence Creator** – Create safety zones with entry/exit alerts  
- 🚨 **SOS Emergency Button** – One-tap alert with dramatic simulation  
- 📡 **Share My Location** – Instant sharing with visual feedback  
- 📱 **100% Mobile-First** – Perfect on phones, tablets & desktop  
- 🌐 **SEO Optimized** – Full meta tags, keywords & lightning-fast performance  

## 🛠️ Technologies Used
- 🔨 **HTML5** – Semantic structure + Leaflet map integration  
- 🎨 **CSS3** – Modern dark safety-themed UI with neon accents  
- 📱 **Bootstrap 5** – Fully responsive & clean interface  
- ⚡ **JavaScript + jQuery** – Real-time tracking, geofences & SOS logic  

## 🚀 How to Run
1. Download the 4 files into the **same folder**  
2. Open `index.html` in any modern browser (allow location if prompted)  
3. Start tracking family & friends instantly!
